# ACAS Xu Isomorphic Networks Benchmark

## Overview
This repository contains the ACAS Xu Isomorphic Networks Benchmark, which is designed to evaluate neural network verification tools. It focuses on checking the bounds of differences between an original ACAS Xu neural network and a perturbed (isomorphic) version of the same network. The benchmark applies a small uniform perturbation to the network weights (`P = 0.001`) and tests whether the outputs of the two networks remain within a specified distance (`EPS = 0.05`).

## Prerequisites
To run the scripts in this repository, you need Python installed along with the following packages:
- `numpy`
- `onnx`

You can install these dependencies using `pip`:
```bash
pip install -r requirements.txt
```

## Usage Instructions
To generate the benchmark instances, run the `generate_properties.py` script. You must provide a numeric integer as a random seed.

```bash
python generate_properties.py <random_seed>
```
Example:
```bash
python generate_properties.py 42
```

This script will:
1. Select an original ACAS Xu ONNX model from the `onnx/original/` directory.
2. Apply a small uniform perturbation to its weights to create an isomorphic network.
3. Save the perturbed network to the `onnx/perturbed/` directory.
4. Generate the corresponding `.vnnlib` specification in the `vnnlib/` directory.
5. Create an `instances.csv` file that lists the pairs of original and perturbed models, their corresponding specification file, and the verification timeout value (default 100 seconds).

## Directory Structure
- `generate_properties.py`: The main script to generate perturbed networks and VNN-LIB specification files.
- `instances.csv`: A CSV file listing the benchmark instances (network pairs, VNN-LIB properties, and timeout).
- `requirements.txt`: Python package dependencies.
- `onnx/`: Contains the ONNX models.
  - `onnx/original/`: The original, unmodified ACAS Xu networks.
  - `onnx/perturbed/`: The resulting perturbed (isomorphic) networks generated by the script.
- `python_scripts/`: Contains auxiliary Python scripts such as those for creating specifications and perturbing the network.
- `vnnlib/`: Contains the generated VNN-LIB specification files for the benchmark properties.
